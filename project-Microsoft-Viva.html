Project Overview

This project simulates Microsoft Viva Engage environments to assess enterprise digital community health. Our team developed an AI-powered health scoring system using sentiment analysis, civility modeling, and custom-designed Community Health Index (CHI) metrics.

Key Contributions

  • Scraped and processed over 10,000 Reddit posts simulating enterprise collaboration
  • Built sentiment and empathy models using GPT to classify civility and tone
  • Designed CHI scoring logic with metrics like reply depth, response latency, ghost posts
  • Created interactive Power BI dashboards to visualize trends and track team culture

Technologies Used

  • Python (Pandas, NLTK, OpenAI API)
  • Power BI
  • Prompt Engineering & GPT-based moderation
