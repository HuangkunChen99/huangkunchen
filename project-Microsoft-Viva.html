Project Overview

This project involved building an AI-powered health scoring system for enterprise-level communities using Reddit datasets. We combined Natural Language Processing (NLP) and Power BI dashboards to analyze sentiment, responsiveness, empathy, and engagement dynamics.

Key Components

  • Reddit API data extraction from HR/People Analytics forums
  • Sentiment and civility scoring via GPT-4 and VADER
  • Back-and-forth engagement measurement (thread depth, ghost posts)
  • Power BI dashboard visualizing overall community health metrics

Tools Used

Python (Pandas, PRAW, NLTK, OpenAI), Power BI, GPT-4 API, Excel
