Microsoft Viva Insight

AI-powered community health scoring for enterprise collaboration platforms.

Project Overview

This project simulates the Microsoft Viva Engage environment and explores health trends in enterprise collaboration platforms. Our team built a system using Reddit data, sentiment analysis, and Community Health Index (CHI) to assess both quantitative and qualitative aspects of engagement.

Key Contributions

  • Scraped and processed over 10,000 Reddit posts from relevant enterprise tech communities.
  • Built sentiment and empathy models using GPT to analyze tone and civility of posts.
  • Designed CHS and CHI scoring systems to track user health, responsiveness, and emotional tone.
  • Created interactive dashboards with Power BI for management to monitor engagement health over time.

Technologies Used

  • Python (Pandas, NLTK, OpenAI API)
  • Power BI
  • BeautifulSoup, Reddit API
  • Microsoft Viva context simulation
